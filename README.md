# Champs gravitational


## Controls:

Mouse :
camera rotation (first person) Hold right click
W / A / S / D :
camera horizontal movement (across the y plane)
Q / E :
camera vertical movement (along the y axis)

The solar system scene contains:

The solar system with all planets except Uranus and Neptune

The AsteroidField scene contains: 

A bunch of asteroids going in random directions

In both scene, there is the same UI.

What you can do in the UI:
-Track a celestial body by clicking on it and enter in orbit mode
When in orbit mode:
    -Display vector field (2D/3D)
    -Modify density and size of the vector field
    -Display field line
    -Check celestials body information(mass, position, speed, ...)
    -Exit orbit mode by clik in the void
-Pause
-Change scene
-Add a celestial body to the scene
